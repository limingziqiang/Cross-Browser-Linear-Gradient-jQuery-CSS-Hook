Support limitations:
  • Degrees (90deg) currently don't work properly in SVG (IE9) gradients - they strech, so the angle is only correct for squares
  • Degrees are not supported in the legacy Webkit (could work around it with percentage, but that's not worth the bytes)
  • Color names (red, green, blue...) are not supported
  • The MS filter only supports 2 colors and top or left, so it will fall back to top and the first and last color if parameter are not supportet
